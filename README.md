# QuizMaker-Pythton
This project makes MCQ/TrueFalse Quizes from parsing JSON files in Python.

# Requirements
 - Python 3.6 or up
 
 # Instructions To Use
 - Make sure all files should be in the same folder before running.
 
 - **Windows**
   ```
   python QuizMaker.py
   ```
 - **Mac or Linux**
   ```
   python3 QuizMaker.py
   ```
